
# QuantEdge XAU/USD

QuantEdge is an explainable quantitative strategy analysis engine designed for XAU/USD Spot Gold.

The system fetches XAU/USD market data and analyzes market conditions using technical and quantitative indicators.

## Strategy Indicators

- EMA 20 and EMA 50
- Relative Strength Index (RSI)
- MACD
- Average True Range (ATR)
- 2-Hour Price Momentum

## Quantitative Strategy Engine

The strategy generates a quantitative score from -100 to +100.

Based on the Quant Score, the model generates:

- STRONG BUY
- BUY
- WAIT
- SELL
- STRONG SELL

## Strategy Levels

When fresh market data and a valid model signal are available, QuantEdge calculates model-generated strategy levels:

- Entry Reference
- Stop Loss
- Target 1
- Target 2

The strategy levels are calculated using ATR-based volatility measurements.

## Explainable Strategy

QuantEdge explains why a signal was generated by analyzing:

- EMA trend direction
- RSI momentum
- MACD confirmation
- Price momentum

## Market Data

Instrument: XAU/USD Spot Gold

Timeframe: 15 Minutes

Timezone: Indian Standard Time (IST)

Market data is retrieved using the Twelve Data API.
